# Business Card

Dit is een basis applicatie, waarin ik een simpele business card heb gemaakt.
Deze opdracht komt uit de Kotlin Course van Google. [Unit 1, Pathway 3](https://developer.android.com/courses/pathways/android-basics-compose-unit-1-pathway-3). De reden dat ik met deze course begonnen ben is om meer te leren te krijgen over hoe dat je apps bouwt in Android Studio. Ik wil de basis kennen, zodat ik een goed oordeel kan geven over de leercurve van Kotlin in combinatie met Android Studio.

---

## 🚀 Features
- Je kan contact informatie uit de app ophalen van de eigenaar van de Business Card

---

## 🛠️ Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose
